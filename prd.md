# PRD Portfolio Website Interaktif — Andy Siswanto A.B

## 1. Ringkasan Produk

Website ini adalah portfolio personal interaktif untuk Andy Siswanto A.B, seorang Web Programmer / Web Developer dengan pengalaman dalam pengembangan website, sistem internal perusahaan, CodeIgniter, database, dan IT support.

Website dibuat dengan tampilan modern, futuristik, interaktif, dan profesional. Gaya visual mengarah ke portfolio developer modern dengan elemen dark mode, animasi partikel, efek glow, scroll animation, card interaktif, dan nuansa teknologi.

Tujuan utama website ini adalah menampilkan profil, pengalaman, skill, sertifikasi, project, dan kontak Andy secara lebih menarik dibandingkan CV biasa, sehingga cocok digunakan untuk melamar kerja, personal branding, dan menawarkan jasa pembuatan website / sistem digital.

---

## 2. Tujuan Produk

### Tujuan Utama

Membuat website portfolio yang mampu memperkenalkan Andy sebagai Web Programmer yang siap bekerja, mampu membangun sistem digital, dan memiliki pengalaman nyata di dunia kerja.

### Tujuan Bisnis / Personal Branding

* Meningkatkan kesan profesional saat dilihat recruiter, HRD, klien, atau calon partner.
* Menampilkan pengalaman kerja dan skill secara visual, modern, dan mudah dipahami.
* Menjadi halaman utama untuk personal branding Andy di internet.
* Menjadi media promosi jasa pembuatan website, aplikasi sederhana, dashboard admin, dan sistem digital.
* Menampilkan kesan bahwa Andy mengikuti perkembangan teknologi modern.
